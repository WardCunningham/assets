Federated Wiki Assets

Here we share from GitHub pages assets that are alredy available via http from various sites owned by Ward Cunningham.

Examples

Consider the Cube of Justice wiki page that includes a Frame script animating a glowing cube coded in ThreeJS. This is served from wiki as an html script cube.html and again as index.html for use as the site's custom home page cube.fed.wiki). Both of these files are again served here on GitHub over https as cube.html and index.html.
